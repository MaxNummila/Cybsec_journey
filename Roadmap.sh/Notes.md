# Notes from Roadmap.sh

## Fundamental IT skills
### NFC
-	Very close proximity PAN
-	Works by transmitting data slowly over radio waves
-	Two different modes: Active communication mode and passive communication mode
-	Active communication mode works automatically when two NFC-compatible devices are brought within proximity to each other.
-	Passive communication mode works by having one active device that can read data from an NFC tag. Once brought into proximity to the tag it generates a carrier field which the tag modulates.
### Bluetooth
-	Short-range wireless connection
-	Works over the low-power GHz spectrum
-	Common Bluetooth attacks:
o	Bluesnarfing – Incorrect installation of drivers leads to an attacker accessing any files on the device
o	Bluejacking – An attacker uses spam advertising to hijack another device
o	Bluebugging – An attacker uses a covert Bluetooth connection to acquire backdoor access to a device, allowing them to eavesdrop on the victim.
### Infrared
-	Used for data transfer over short-to-mid ranges
-	Infrared misusage:
o	Infrared smart light bulbs are an ideal target for attackers to spoof all connected devices on one Wi-Fi network.
o	Intercepting infrared waves to access security cameras

## Operating Systems
### Windows
-	Installation requires: USB flash drive, Internet connection, Product key (Optional)
-	Troubleshooting Common issues
o	PC won’t boot from USB – Fast boot and Secure boot need to be correctly configured, possibly a faulty USB
o	Windows cannot be installed to this disk – Drive partition misconfiguration is the most common reason
